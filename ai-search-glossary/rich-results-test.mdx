## Definition

The Rich Results Test is Google's free tool for validating structured data markup and previewing how a page may appear as a rich result in Google Search. It identifies schema markup errors, warnings, and eligibility for specific rich result types.

## Why It Matters for AI Search

The Rich Results Test is the structured data audit starting point. Before structured data can contribute to AI citation, it must be correctly implemented and error-free. Running key pages through the Rich Results Test identifies implementation errors that would prevent schema from being processed — including common issues like missing required properties, incorrect types, and inaccessible markup.
